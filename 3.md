✅ 3.2) Write a program to count the number of vowels in a string (No loop allowed). Using sum and lambda without explicit loops.

s = "Artificial Intelligence"
vowels = "aeiouAEIOU"

count = sum(map(lambda c: c in vowels, s))
print("Number of vowels:", count)
